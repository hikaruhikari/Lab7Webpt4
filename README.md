3. Manfaat utama penggunaan View Layout dalam CodeIgniter 4 adalah:


- Konsistensi Tampilan: Memastikan elemen dasar seperti header, navigation, dan footer tetap sama di setiap halaman tanpa perlu menulis ulang kodenya.


- Efisiensi Perubahan: Jika ada perubahan pada struktur dasar (misal menambah menu navigasi), Anda cukup mengubah satu file layout saja (main.php), dan perubahan tersebut akan langsung diterapkan di seluruh halaman yang menggunakan layout tersebut

4. perbedaan view biasa dengan view cell

- view biasa dengan File template standar untuk menampilkan data dari controller. sedangkan view cell dengan Komponen UI modular yang dapat digunakan kembali di berbagai tempat.

- view biasa itu Bergantung pada data yang dikirimkan oleh Controller utama. sedangkan view cell itu Memiliki logika/class sendiri (Cell Class) untuk mengambil data secara mandiri.

- view biasa Digunakan untuk konten utama sebuah halaman. sedangkan view cell Digunakan untuk elemen pendukung seperti sidebar, widget, atau menu.
